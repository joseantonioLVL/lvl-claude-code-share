# Claude Code — Guía Victoria Lab

Guía completa de Claude Code en español para el equipo de Victoria Lab (~30 personas). Publicada en GitHub Pages.

## Publicar en GitHub Pages

1. Sube este repositorio a GitHub
2. Ve a Settings → Pages
3. Source: `main` branch, carpeta raíz (`/`)
4. Guarda — GitHub Pages se activa en ~1 minuto

El sitio es un único archivo `index.html` con todo el CSS y JS inline. No requiere build ni dependencias.

## Datos de ejemplo

La guía usa fragmentos reales del proyecto **Whoop Screen Exploration Q4 2025** como casos de uso ilustrativos. Los datos crudos no están incluidos en este repositorio (ver `.gitignore`).
